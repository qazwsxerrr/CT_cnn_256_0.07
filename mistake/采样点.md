下面是一份**简洁大纲**，专门说明这次“Radon 时域采样点 (s_j) 与频域采样点 (\xi_j) 对齐”的改动，以及对应的代码实现思路。

------

## 1. 改动目标（一句话）

让：

- **Radon 时域采样点** (s_j) 的步长
- 与理论中频域采样点
   $$
   \xi_j = \frac{2\pi j}{N},|\beta|_2
   $$
   通过关系
   $$
   \xi_j = \frac{2\pi j}{N,\Delta s}
   $$
   **严格一致**，从而保证：

$$
 \Delta s = \frac{1}{|\beta|*2},\quad s_j = \frac{\kappa*{\min}+j}{|\beta|_2}.
 $$

------

## 2. 理论：时域 (s_j) 与频域 (\xi_j) 的一致性

1. **时域（Radon 变量）采样：**
   - 方向：$$\alpha = \beta / |\beta|_2$$
   - 投影变量：$$s = \alpha \cdot x$$
   - 对应系数网格 $$\beta\cdot k \in [\kappa_{\min}, \kappa_{\max}]$$，长度 $$N = \kappa_{\max}-\kappa_{\min}+1$$
   - **定义时域采样点：**
      $$
      s_j = \frac{\kappa_{\min}+j}{|\beta|_2},\quad j=0,\dots,N-1
      $$
      => 步长 $$\Delta s = 1/|\beta|_2$$。
2. **离散 FFT 与频域采样：**
   - 对 $$\mathcal R_\alpha f(s_j)$$ 做长度 $$N$$ 的 FFT，标准关系：
      $$
      \xi_j = \frac{2\pi j}{N,\Delta s}
      $$
   - 代入 $$\Delta s = 1/|\beta|_2$$：
      $$
      \xi_j = \frac{2\pi j}{N},|\beta|_2
      $$
   - 正好与理论推导中 (3.1) 的频域采样一致。

> 结论：**只要 Radon 时域步长取 $$\Delta s = 1/|\beta|_2$$，就自然得到正确的 $$\xi_j$$，无需再在代码里额外乘 $$|\beta|_2$$。**

------

## 3. 代码改动要点

### 3.1 Radon 时域采样点 (s_j)（ODL 几何）

原来一般写法（步长=1，有问题）：

```python
# 原实现：步长 Δs = 1
det_partition = odl.uniform_partition(
    -self.n_detectors / 2,
     self.n_detectors / 2,
     self.n_detectors
)
```

**改为与理论一致的采样：**

```python
# 假设已知 beta, E_plus → kappa_min, kappa_max
beta = np.array([1.0, 21.0], dtype=np.float64)
beta_norm = np.linalg.norm(beta)

kappa_min, kappa_max = 0, 440          # 当前配置下的例子
N = kappa_max - kappa_min + 1          # = 441

s_min = kappa_min / beta_norm          # 0 / ||β||
s_max = kappa_max / beta_norm          # 440 / ||β||

det_partition = odl.uniform_partition(
    s_min,   # 起点 s_0
    s_max,   # 终点 s_{N-1}
    N        # 采样点个数
)
```

这样：

- Radon 投影的离散点就是
   $$s_j = (\kappa_{\min} + j)/|\beta|_2$$，步长 $$\Delta s = 1/|\beta|_2$$。

### 3.2 频域采样点 (\xi_j)（构造 Φ、G）

Radon 时域正确以后，FFT 自动给出正确的频率：

- `torch.fft.fft(radon_data, dim=-1)` 对应的连续频率就是
   $$
   \xi_j = \frac{2\pi j}{N,\Delta s}
   = \frac{2\pi j}{N},|\beta|_2
   $$

因此在 `FourierOperatorCalculator` 构造 Φ 和 G 时，**只需用和 FFT 相同的公式**：

```python
N = self.n_coefficients
j = torch.arange(N, dtype=torch.float64)

# 与 Radon 采样 $$\Delta s = 1/||\beta||$$ 对应
xi = 2.0 * np.pi * j / N * self.beta_norm.item()

ax, ay = self.alpha.double().tolist()
# 频域基函数 $$\hat{\phi}(\xi_j \alpha)$$
B2 = bspline.B2_hat_complex(ax * xi.numpy())
B1 = bspline.B1_hat_complex(ay * xi.numpy())
Phi = B2 * B1  # (N,)

# $$G_{j,n} = \exp(-i \xi_j \alpha\cdot k_n)$$
xs = torch.arange(width, dtype=torch.float64)
ys = torch.arange(height, dtype=torch.float64)
X, Y = torch.meshgrid(xs, ys, indexing='xy')
kdot = (ax * X + ay * Y).reshape(-1)
phase = torch.outer(xi, kdot)
G = torch.exp(-1j * phase)
```

这里的 `xi` 与 Radon+FFT 生成 $$F_{\text{obs}}$$ 的频率完全一致，不再有刻度偏差。

------

## 4. 最小的检查步骤（sanity check）

1. 随机生成一组系数矩阵 $$c$$；
2. **物理链路**：$$c \to f \to \mathcal R \to \text{FFT}$$ 得到 $$F_{\text{phys}}$$；
3. **理论链路**：$$c \to d \to \Phi G d$$ 得到 $$F_{\text{model}}$$；
4. 比较相对误差：
    $$
    \frac{|F_{\text{phys}} - F_{\text{model}}|*2}{|F*{\text{phys}}|_2}.
    $$

在无噪声、实现正确的情况下，误差应显著下降，说明**时域/频域刻度已对齐**。
