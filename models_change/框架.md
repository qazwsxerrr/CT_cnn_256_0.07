这是一个将传统的 **TV正则化求解** 替换为 **基于物理模型的深度学习（Learned Gradient Descent, LGD）** 的完整实现方案。

该方案的核心思想是：利用神经网络（CNN）来替代传统迭代算法（如ADMM或梯度下降）中的人工设计更新步长和正则化投影步骤，同时保留物理模型（Radon变换和傅里叶变换）的精确性。

### 实现步骤详解

#### 第一步：构建物理算子层（处理复数的核心）

**目标**：实现 $\mathbf{A}\mathbf{x} = \mathbf{y}$ 及其伴随 $\mathbf{A}^H$。
**复数处理策略**：

1.  \*\*正向传播 ($\mathbf{A}$) \*\*：输入实数向量 $\mathbf{x}$ $\rightarrow$ FFT（变复数）$\rightarrow$ 乘以 $\mathbf{\Phi}$（复数）$\rightarrow$ 输出复数观测 $\mathbf{y}$。
2.  **伴随传播 ($\mathbf{A}^H$)**：输入复数残差 $\mathbf{r}$ $\rightarrow$ 乘以 $\mathbf{\Phi}^*$（复数）$\rightarrow$ IFFT（仍为复数）$\rightarrow$ **取实部 (.real)** $\rightarrow$ 输出实数梯度。
    *这一步是“复数隔离墙”，确保网络只接收实数梯度。*

#### 第二步：设计 1D CNN 网络

**目标**：替代 `model.py` 中的 2D CNN，因为本问题求解的是 1D 系数向量 $\mathbf{d}$。
**输入**：堆叠的实数张量，包含：

  * 当前估计值 $\mathbf{x}$
  * 数据项梯度 $\nabla_{\text{data}}$ (来自物理算子)
  * 正则项梯度 $\nabla_{\text{reg}}$ (通常取 $\lambda \mathbf{x}$ 或 0，由网络学习复杂正则化)
  * 记忆状态 $\mathbf{s}$ (Memory state)

#### 第三步：构建展开（Unrolled）优化循环

**目标**：模拟梯度下降的迭代过程。
**流程**：

1.  初始化 $\mathbf{x}^{(0)}$ 和记忆 $\mathbf{s}^{(0)}$。
2.  进入循环 $k = 1 \dots N$：
      * 计算复数残差：$\mathbf{r} = \mathbf{A}\mathbf{x}^{(k-1)} - \mathbf{y}_{\text{obs}}$
      * 计算实数数据梯度：$\mathbf{g}_{\text{data}} = \mathbf{A}^H \mathbf{r}$
      * 计算正则梯度参考：$\mathbf{g}_{\text{reg}} = \lambda \mathbf{x}^{(k-1)}$
      * 将 $\{\mathbf{x}, \mathbf{g}_{\text{data}}, \mathbf{g}_{\text{reg}}, \mathbf{s}\}$ 拼接到一起输入 CNN。
      * CNN 输出更新量 $\delta \mathbf{x}$ 和新状态 $\mathbf{s}$。
      * 更新：$\mathbf{x}^{(k)} = \mathbf{x}^{(k-1)} - \delta \mathbf{x}$。

### 修改后的代码实现 (`model_lgd_1d.py`)

```python
import torch
import torch.nn as nn
import numpy as np

# ==========================================
# 1. 物理算子层：封装Radon与傅里叶变换
#    (复数运算被限制在此类内部)
# ==========================================
class RadonFourierOperator(nn.Module):
    def __init__(self, N, m=2):
        """
        N: 采样点数/向量长度
        m: B-spline阶数
        """
        super().__init__()
        self.N = N
        
        # 构造频率坐标 [0, 2pi/N, ..., -2pi/N] 以匹配 PyTorch FFT 顺序
        k = torch.fft.fftfreq(N, d=1.0/N)
        w = 2 * np.pi * k
        
        # 计算 B-spline 的频域响应 Phi (复数对角矩阵)
        # 公式: (sin(w/2)/(w/2))^m * exp(-i*m*w/2)
        w_safe = w.clone()
        w_safe[w_safe == 0] = 1e-9
        magnitude = (torch.sin(w_safe / 2) / (w_safe / 2)) ** m
        magnitude[w == 0] = 1.0
        phase = torch.exp(-1j * m * w / 2)
        phi_diag = magnitude * phase
        
        # 注册为 Buffer，不作为可训练参数，但随模型移动到 GPU
        self.register_buffer('phi_diag', phi_diag)

    def forward(self, x):
        """
        正向投影 A(x): 实数域 -> 复数频域
        x: (Batch, 1, N) 实数
        Return: (Batch, 1, N) 复数
        """
        # 1. FFT (实 -> 复)
        x_freq = torch.fft.fft(x, dim=-1)
        # 2. 乘以对角矩阵 Phi
        y = self.phi_diag * x_freq
        return y

    def adjoint(self, y):
        """
        伴随投影 A^H(y): 复数频域 -> 实数域
        用于计算数据保真项梯度
        y: (Batch, 1, N) 复数残差
        Return: (Batch, 1, N) 实数梯度
        """
        # 1. 乘以 Phi 的共轭
        x_freq = torch.conj(self.phi_diag) * y
        # 2. IFFT
        x_complex = torch.fft.ifft(x_freq, dim=-1)
        # 3. 【关键步骤】取实部，舍弃虚部噪音，回到实数域
        # 乘以 N 是为了匹配梯度的量级 (视FFT定义而定，这里作为Scaling因子)
        return x_complex.real * self.N

# ==========================================
# 2. CNN 网络：替换原 Conv2d 为 Conv1d
# ==========================================
class CNN_1D(nn.Module):
    def __init__(self, n_memory):
        super(CNN_1D, self).__init__()
        # 输入通道: x(1) + grad_data(1) + grad_reg(1) + memory(n_memory)
        input_channels = 3 + n_memory
        
        self.conv1 = nn.Conv1d(input_channels, 32, 3, padding=1)
        self.conv2 = nn.Conv1d(32, 32, 3, padding=1)
        # 输出通道: update_step(1) + new_memory(n_memory)
        self.conv3 = nn.Conv1d(32, 1 + n_memory, 3, padding=1)
        self.relu = nn.ReLU()

        # 权重初始化
        nn.init.xavier_uniform_(self.conv1.weight)
        nn.init.constant_(self.conv1.bias, 0.01)
        nn.init.xavier_uniform_(self.conv2.weight)
        nn.init.constant_(self.conv2.bias, 0.01)
        nn.init.xavier_uniform_(self.conv3.weight)
        nn.init.constant_(self.conv3.bias, 0.0)

    def forward(self, x):
        x = self.relu(self.conv1(x))
        x = self.relu(self.conv2(x))
        x = self.conv3(x)
        return x

# ==========================================
# 3. Learned Gradient Descent 主模型
# ==========================================
class LearnedGD_Recon(nn.Module):
    def __init__(self, N, n_iter=10, n_memory=5):
        super(LearnedGD_Recon, self).__init__()
        self.n_iter = n_iter
        self.n_memory = n_memory
        
        # 物理算子
        self.operator = RadonFourierOperator(N)
        
        # 优化器网络
        self.cnn = CNN_1D(n_memory)
        
        # 可学习的正则化权重 (替代手动调节 TV 参数)
        self.reg_lambda = nn.Parameter(torch.tensor(0.01))

    def forward(self, y_obs, x_init=None):
        """
        y_obs: (Batch, 1, N) 复数频域观测数据 [F矩阵]
        x_init: (Batch, 1, N) 初始猜测 (可选)
        """
        batch_size = y_obs.shape[0]
        N = y_obs.shape[-1]
        
        # 初始化 x (实数)
        if x_init is None:
            x = torch.zeros(batch_size, 1, N, device=y_obs.device)
        else:
            x = x_init

        # 初始化记忆状态 s
        s = torch.zeros(batch_size, self.n_memory, N, device=y_obs.device)
        
        # 记录迭代轨迹
        x_history = [x]

        for i in range(self.n_iter):
            # --- A. 数据保真项梯度 (Data Fidelity) ---
            # 1. 正向投影: A x (结果为复数)
            y_pred = self.operator(x)
            
            # 2. 计算残差: A x - y (复数减法)
            residual = y_pred - y_obs
            
            # 3. 伴随投影: A^H (residual) (结果为实数)
            grad_data = self.operator.adjoint(residual)
            
            # --- B. 正则化梯度 (Regularization) ---
            # 这里使用简单的 L2 梯度作为特征，CNN 会学会如何将其变为 TV 或其他复杂的正则项
            grad_reg = self.reg_lambda * x 

            # --- C. CNN 更新 ---
            # 拼接所有特征 (全部为实数)
            cnn_input = torch.cat([x, grad_data, grad_reg, s], dim=1)
            
            cnn_output = self.cnn(cnn_input)
            
            # 分离输出
            dx = cnn_output[:, 0:1, :]   # 更新步长
            s_new = torch.relu(cnn_output[:, 1:, :]) # 新的记忆状态
            
            # --- D. 梯度下降步骤 ---
            # x_{k+1} = x_k - Network(x_k, grad_k, ...)
            # 注意：网络可以直接输出 -dx，也可以输出 dx，这里假设是负梯度方向
            x = x - dx 
            s = s_new
            
            x_history.append(x)

        return x, x_history

# ==========================================
# 4. 测试代码 (演示维度和复数处理)
# ==========================================
if __name__ == "__main__":
    device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
    N = 64
    
    # 实例化模型
    model = LearnedGD_Recon(N=N, n_iter=5).to(device)
    
    # 模拟输入数据
    # 1. 观测数据 y_obs 是复数 (Radon频域采样)
    y_fake = torch.randn(2, 1, N, dtype=torch.cfloat).to(device)
    
    # 2. 前向推断
    x_est, history = model(y_fake)
    
    print(f"观测数据 (y) 类型: {y_fake.dtype}") # torch.complex64
    print(f"重建结果 (x) 类型: {x_est.dtype}")  # torch.float32
    print(f"重建结果维度: {x_est.shape}")       # [2, 1, 64]
    print(f"迭代步数: {len(history)-1}")
```

### 代码中的关键点总结：

1.  **`RadonFourierOperator` 类**：替代了 ODL。它利用 `torch.fft` 和预计算的 `phi_diag` 实现了文件1中的矩阵乘法。
2.  **`.real` 操作**：在 `adjoint` 函数中，`x_complex.real` 是最重要的一行，它将复数域的计算结果“投影”回实数域，使得后续的 CNN 可以处理。
3.  **`CNN_1D`**：将原有的 2D 卷积层换成了 1D，因为我们是在求解系数向量 $\mathbf{d}$。
4.  **输入特征**：除了传统的梯度 $\nabla f(\mathbf{x})$，还加入了 $\mathbf{x}$ 本身和记忆 $\mathbf{s}​$，这使得网络不仅仅是学习一个步长，而是学习了一个完整的优化算法（类似于动量法或 Adam）。
