# 神经网络梯度输入修改说明文档





## 1. 理论基础与梯度公式



为了将物理约束引入深度网络（Learned Gradient Descent, LGD），我们需要计算两个关键的梯度项，并将其作为通道（Channel）拼接到 CNN 的输入中。



### 1.1 数据拟合项梯度 (Data Fidelity Gradient)



目标函数：$J_{data}(\mathbf{d}) = \| A \mathbf{d} - \mathbf{F} \|_2^2$

其中 $A = \Phi G$，$\mathbf{d}$ 为实数系数向量，$\mathbf{F}$ 为复数频域观测值。

梯度公式（关于实数 $\mathbf{d}$）：



$$\nabla_{\mathbf{d}} J_{data} = 2 \cdot \text{Re} \{ A^H (A \mathbf{d} - \mathbf{F}) \}$$

- **前向 (Forward)**: $F_{pred} = \Phi \cdot (G \cdot \mathbf{d})$
- **伴随 (Adjoint/Backward)**: $\nabla = G^H \cdot (\Phi^H \cdot \text{Residual})$
- **实部提取**: 因为神经网络处理的是实数 B-样条系数，最后必须取实部 $\text{Re}\{\cdot\}$。



### 1.2 正则化项梯度 (Dirichlet Energy Gradient)



目标函数：$J_{reg}(\mathbf{d}) = \frac{1}{2} \int \| \nabla \mathcal{I}(\mathbf{d}) \|^2 \approx \frac{1}{2} \mathbf{d}^T L \mathbf{d}$

其中 $L$ 是拉普拉斯矩阵。

梯度公式：



$$\nabla_{\mathbf{d}} J_{reg} = - \Delta \mathbf{d}$$



(注：为了最小化能量，梯度下降方向为 $+\Delta \mathbf{d}$，即热传导方程/平滑方向。梯度本身指向能量增加的方向，即 $-\Delta \mathbf{d}$)。

在离散图像/矩阵中，这可以通过卷积实现：



$$\nabla_{\mathbf{d}} J_{reg} \approx \text{Conv2d}(\mathbf{d}, K_{laplace})$$



使用标准的拉普拉斯卷积核 $K_{laplace} = [[0, -1, 0], [-1, 4, -1], [0, -1, 0]]$。

### 2.1 修改 `RadonFourierOperator2D`



**目的**：确保算子严格执行 $A = \Phi G$ 的矩阵运算，并支持 PyTorch 的自动微分。

Python



```
# [model.py] 替换原有的 RadonFourierOperator2D 类

class RadonFourierOperator2D(nn.Module):
    """
    实现基于 F = Phi G d 的频域算子。
    严格对应算法说明文档中的矩阵定义。
    """
    def __init__(self, beta=(1, 21), height=21, width=21, m=2, normalize=True):
        super().__init__()
        self.height = height
        self.width = width
        self.N = height * width
        
        # 使用 FourierOperatorCalculator 获取 Phi 和 G 矩阵
        # 引用 中的 FourierOperatorCalculator 用法
        self.fourier_calc = FourierOperatorCalculator(
            beta=beta, n_coefficients=self.N, m=m
        )
        
        # 注册缓冲区 (Buffer) 以便随模型保存并在 GPU 上运行
        # Phi是对角阵 (N,)，G是稠密阵 (N, N)
        # 注意：为了神经网络的输入稳定性，这里使用 complex64 (单精度复数)
        # 如果需要极高精度验证，可以改为 complex128，但会增加显存消耗
        self.register_buffer('Phi', self.fourier_calc.Phi_diagonal.to(torch.complex64))
        self.register_buffer('G', self.fourier_calc.G_matrix_2d.to(torch.complex64))
        
        # 预计算共轭转置，加速梯度计算
        self.register_buffer('Phi_H', self.Phi.conj())
        self.register_buffer('G_H', self.G.conj().t())
        
        # 索引映射 (用于处理 2D <-> 1D 转换)
        self.register_buffer('flatten_order', self.fourier_calc.flatten_order.to(torch.long))
        self.register_buffer('unflatten_order', self.fourier_calc.unflatten_order.to(torch.long))

        # 归一化因子 (可选)
        self.scale_factor = 1.0
        if normalize:
            # 简单估算算子范数用于归一化
            with torch.no_grad():
                # Power iteration creation of norm
                x = torch.randn(self.N, dtype=torch.complex64)
                for _ in range(5):
                    gx = torch.matmul(x, self.G.t())
                    ax = gx * self.Phi
                    phix = ax * self.Phi_H
                    atx = torch.matmul(phix, self.G_H.t())
                    norm = torch.norm(atx)
                    x = atx / norm
                self.scale_factor = 1.0 / torch.sqrt(norm).item()

    def forward(self, coeff_matrix: torch.Tensor) -> torch.Tensor:
        """
        前向传播: d (2D) -> F_pred (1D Complex)
        公式: F = Phi * (G * d_flat)
        """
        batch_size = coeff_matrix.shape[0]
        
        # 1. 展平并重排序 d (B, 1, H, W) -> (B, N)
        coeff_flat = coeff_matrix.view(batch_size, -1)
        # d 是实数，但在频域计算中视为复数实部
        d_ordered = coeff_flat[:, self.flatten_order].to(self.G.dtype)
        
        # 2. 矩阵乘法 G * d
        # (B, N) @ (N, N)^T -> (B, N)
        Gd = torch.matmul(d_ordered, self.G.t())
        
        # 3. 对角矩阵乘法 Phi * Gd
        F_pred = Gd * self.Phi.unsqueeze(0)
        
        return F_pred * self.scale_factor

    def adjoint(self, residual: torch.Tensor) -> torch.Tensor:
        """
        伴随算子 (用于计算梯度): Residual (1D Complex) -> Grad_d (2D Real)
        公式: Grad = Re{ G^H * (Phi^H * Residual) }
        """
        # 1. 处理维度
        if residual.dim() == 3 and residual.shape[1] == 1:
            residual = residual.squeeze(1)
        
        # 2. 对角矩阵乘法 Phi^H * R
        res_norm = residual * self.scale_factor
        PhiH_R = res_norm * self.Phi_H.unsqueeze(0)
        
        # 3. 矩阵乘法 G^H * (Phi^H * R)
        d_grad_complex = torch.matmul(PhiH_R, self.G_H.t())
        
        # 4. 取实部并映射回 2D 空间
        # 注意：这里只取实部，因为物理量 d 是实数
        d_grad_real = d_grad_complex.real
        
        # 反向映射排序
        batch_size = residual.shape[0]
        grad_output = torch.zeros_like(d_grad_real)
        grad_output[:, self.flatten_order] = d_grad_real
        
        return grad_output.view(batch_size, 1, self.height, self.width)
```



### 2.2 修改 `TheoreticalGradientDescent`



**目的**：正确实现数据项梯度的调用，并修正迪利克雷梯度的卷积核定义。

Python



```
# [model.py] 替换原有的 TheoreticalGradientDescent 类

class TheoreticalGradientDescent(nn.Module):
    def __init__(self, beta=(1, 21), height=21, width=21,
                 regularizer_type='dirichlet', lambda_reg=0.01):
        super().__init__()
        self.operator = RadonFourierOperator2D(beta, height, width)
        self.regularizer_type = regularizer_type
        self.lambda_reg = lambda_reg
        self.step_size = 0.1 # 这里的 step_size 仅用于纯理论 GD，CNN 中会学习步长

        # 定义迪利克雷能量梯度的卷积核 (Negative Laplacian)
        # 对应梯度方向 (指向能量增加的方向)
        # Gradient of 1/2 ||grad u||^2 is -Laplacian u
        # Kernel approximation for -Laplacian:
        self.register_buffer('laplace_kernel', torch.tensor(
            [[0.0, -1.0, 0.0],
             [-1.0, 4.0, -1.0],
             [0.0, -1.0, 0.0]]
        ).view(1, 1, 3, 3))

    def compute_data_fidelity_gradient(self, coeff_matrix, F_observed):
        """
        计算数据项梯度: 2 * A^H * (A d - F)
        """
        # 1. 前向计算 F_pred
        F_pred = self.operator(coeff_matrix)
        
        # 2. 计算残差
        if F_observed.dim() == 3 and F_observed.shape[1] == 1:
            F_observed = F_observed.squeeze(1)
            
        # 确保 F_observed 类型匹配 (可能从 DataGenerator 传来的双精度，需转单精度)
        F_obs = F_observed.to(dtype=F_pred.dtype)
        residual = F_pred - F_obs
        
        # 3. 伴随算子计算梯度
        # adjoint 方法内部已经包含了取实部和 reshape 的操作
        gradient = self.operator.adjoint(residual)
        
        return 2.0 * gradient

    def compute_regularization_gradient(self, coeff_matrix):
        """
        计算正则化梯度。
        对于迪利克雷能量 R(d) = 1/2 ||grad d||^2，梯度为 -Laplacian(d)。
        """
        if self.regularizer_type == 'dirichlet':
            return self._dirichlet_gradient(coeff_matrix)
        elif self.regularizer_type == 'tikhonov':
            return 2 * coeff_matrix # L2 正则梯度
        else:
            return torch.zeros_like(coeff_matrix)

    def _dirichlet_gradient(self, coeff_matrix):
        """
        使用卷积计算拉普拉斯梯度。
        """
        # 使用 replicate padding 处理边界，保持尺寸一致
        padded = F.pad(coeff_matrix, (1, 1, 1, 1), mode='replicate')
        grad = F.conv2d(padded, self.laplace_kernel, padding=0)
        return grad

    def gradient_descent_step(self, coeff_matrix, F_observed):
        # 纯理论 GD 步骤 (用于对比或预训练)
        data_grad = self.compute_data_fidelity_gradient(coeff_matrix, F_observed)
        reg_grad = self.compute_regularization_gradient(coeff_matrix)
        
        total_grad = data_grad + self.lambda_reg * reg_grad
        updated_coeff = coeff_matrix - self.step_size * total_grad
        return updated_coeff
```

## 3. 实现细节总结



1. **精度处理**：
   - 在 `RadonFourierOperator2D` 中，我们显式使用了 `torch.complex64`。虽然《算法说明.md》中提到 Tikhonov 解析解需要双精度，但在神经网络的前向传播中，保留梯度的单精度通常足够，且能大幅提高训练速度。
   - `F_observed` 在输入计算梯度前，被强制转换为了匹配 `F_pred` 的精度 (`dtype=F_pred.dtype`)。
2. **迪利克雷梯度的符号**：
   - 我们定义正则化项为能量 $E$。想要最小化能量，网络需要知道梯度的方向（即能量增加最快的方向），然后反向移动。
   - 代码中 `_dirichlet_gradient` 返回的是正梯度（指向高能量/非平滑状态）。
   - CNN 的更新逻辑是 `coeff_current - learned_update`。CNN 会通过学习 `learned_update`（基于输入的梯度特征）来决定实际的更新步长和方向。
3. **矩阵与卷积的结合**：
   - 数据项使用了全连接性质的矩阵乘法（全局感受野）。
   - 正则化项使用了卷积（局部感受野）。
   - 这种设计结合了 Radon 变换的全局相关性和图像先验的局部平滑性，非常适合 CT 重建任务。
